Sube tu contenido al repositorio

Dentro de tu nuevo repositorio, haz clic en "uploading an existing file" o utiliza Git para subir tus archivos. Para subir con Git:

  • Si aún no lo has hecho, inicializa un repositorio local en la carpeta de tu proyecto con git init.
  • Agrega el repositorio de GitHub como el remoto con git remote add origin https://github.com/{username}/{repository}.git, reemplazando {username} y {repository} con tu nombre de usuario de GitHub y el nombre del repositorio, respectivamente.
    Agrega tus archivos al repositorio local con git add ..
  • Realiza tu primer commit con git commit -m "Initial commit".
  • Sube tus archivos al repositorio de GitHub con git push -u origin master o git push -u origin main, dependiendo de la nomenclatura de tu rama principal.
