Habilita GitHub Pages para tu repositorio

  • Ve a la pestaña "Settings" de tu repositorio en GitHub.
  • Desplázate hacia abajo hasta la sección "GitHub Pages".
  • En "Source", selecciona la rama donde has subido tu contenido (generalmente master o main) y haz clic en "Save".
  • GitHub te proporcionará una URL donde tu sitio estará disponible públicamente, algo como https://{username}.github.io/{repository}/ para proyectos, o https://{username}.github.io/ para sitios de usuario.
